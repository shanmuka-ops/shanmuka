CSS FLEX BOX

  • To make anything side-by-side we have to write inside the parent element like a container.
  • EG:(div class="parent")(/div)
  • Height and width will be ocupie by parent based on the content inside the container.
  • After writing content inside the parent we have to use (display:flex;) in style css to make the content side-by-side.
  • To make gap between the content inside the container we use (justify-content:space-between).
  • Justify-content will on work wile using display:flex;.
  • To align the content or boxes inside the container we use (align-items:center; or flex-top,flex-bottom).
  • To move the items horizontal or vertical we use(justify-content:end or start:).
  • To fit the images inside the container we use (object-fit:cover;).
  • Use bootstrap site to get icons.
  • (flex:wrap;) will be usefull to arrange the content line-by-line based on screen inside the container.
  • use margin:0; to make the contact looks good.
