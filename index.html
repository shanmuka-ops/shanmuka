INTRODUCTION TO HTML

  • HTML(hyper text markup language) its an markup language.
  • HTML is used to design the skeleton of web page.

BASIC HTML BOILER PLATE

  • "!doctype html",it will represent the version of html:5.
  • Here doctype is an element and html is attribute.
  • "html" everything should be inside the html.
  • "head" it was an link holder.
  • "body" it was an content holder.

LIST TYPES AND THERE ELEMENTS

  • "Order list"(OL) it was used to give an order to the list in numerical or alphabatical way.
  • "Unorder list"(UL)it was used to give an order to the list in a way of symbols.
  • Description list"(DL) organizes content where each term(a word,phase,concept) is paired with one or more explanations.
