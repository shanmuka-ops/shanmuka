  • (meta name="view port" content="width-device-width initial scale=1.0") it's to make the same view of document scale in any type of screens.
  • (TITLE) to address the website and it's plays an important role in 'search engine optimization' for a website.
  • (meta name="keywords" content="options") it will allow google to match the keywords to a user while searching.

  • (link:favicon) to make logo to the website to show in tab with title.

TYPES OF CSS

  1. Inline css
  2. Internal css
  3. External css

INLINE CSS

  • It will stay within the line,It should be write inside the body part.
  • (h1 style="color:blue") to change text color.
  • (h1 style="color:blue;background-color:black") used to add background color.

INTERNAL CSS

  • It will stay within the file,It should be write inside the head part.
  • It's also will contain same properties like another two.

EXTERNAL CSS

  • It will stay outside the file,It will be write in head part.
  • (link rel="stylesheet" href="./style.css").

DAY-7

CSS FLEX BOX

  • To make anything side-by-side we have to write inside the parent element like a container.
  • EG:(div class="parent")(/div)
  • Height and width will be ocupie by parent based on the content inside the container.
  • After writing content inside the parent we have to use (display:flex;) in style css to make the content side-by-side.
  • To make gap between the content inside the container we use (justify-content:space-between).
  • Justify-content will on work wile using display:flex;.
  • To align the content or boxes inside the container we use (align-items:center; or flex-top,flex-bottom).
  • To move the items horizontal or vertical we use(justify-content:end or start:).
  • To fit the images inside the container we use (object-fit:cover;).
  • Use bootstrap site to get icons.
  • (flex:wrap;) will be usefull to arrange the content line-by-line based on screen inside the container.
  • use margin:0; to make the contact looks good.
