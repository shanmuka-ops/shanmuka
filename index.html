META ELEMENTS

  • (META CHARSET="UTF-8") it will make the web page to accept any kind off language and symbols to write in you text.
  • (META HTTP-EQUIV="X-UA-COMPATIBLE" CONTENT="IE EDGE") it will allow your document to exigute or open in any browser.
  • (meta name="view port" content="width-device-width initial scale=1.0") it's to make the same view of document scale in any type of screens.
  • (TITLE) to address the website and it's plays an important role in 'search engine optimization' for a website.
  • (meta name="keywords" content="options") it will allow google to match the keywords to a user while searching.
  • (meta name="description" content="") to specify the work of website.
  • (link:favicon) to make logo to the website to show in tab with title.

ANCHOR ELEMENT

  • href="" attribute is used to redirect from one page to another.
  • (a href="url link")site name(/a) for External sites.
  • (a href="/internal page")page name(/a) for internal.
  • (a href="trl:+91 number")call to +91(/a) to call.
  • (a href="http://wa.me/number?text=message")whatsapp us(/a) to whatsapp
  • (a href="mail to:person mail")email us(/a) to send mail.
